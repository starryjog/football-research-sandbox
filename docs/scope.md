# 项目范围

本文定义本项目记录什么、不记录什么，以及样本库和全量库的边界。

## 项目定位

本项目是非官方足球研究资料库和静态站点，聚焦中国青训、亚洲青年赛事、中日韩留洋样本、专题项目和少量对照样本。它服务于资料整理、交叉核验和长期跟踪，不是任何协会、俱乐部、赛事方或数据库供应商的官方发布。

项目里的名单、出场、身价、履历和专题判断都应理解为公开资料整理结果。读者在引用时应回到原始来源复核。

## 当前重点

当前维护重点包括：

- AFC U17 / U23 亚洲杯相关球员与赛事资料。
- 中国、日本、韩国 U17 和 U23 样本。
- 中国青训、学校、俱乐部和专题项目路径。
- 中国、日本、韩国球员留洋历史和当前海外注册样本。
- 五大联赛亚洲教练样本，区分 AFC 成员协会主口径和广义地理边界口径。
- 中国队相关亚洲赛事和 FIFA 青少年赛事归档。

少量乌兹别克斯坦 U17、俱乐部世界杯、中国男足青年教练等数据作为对照或专题补充存在，不代表项目已覆盖这些领域的全量数据。

## 非官方声明

本项目不是：

- 官方球员注册系统。
- 官方报名名单或出场数据发布渠道。
- 官方留洋人数统计。
- 实时比分、实时身价或合同数据库。
- 球探评级、转会建议或商业预测产品。

如项目数据与官方公告、赛事 PDF、俱乐部公告或原始数据库冲突，应以更原始、更接近事件发生方的来源为准，并在项目内记录冲突。

## 样本库 vs 全量库

项目同时包含接近完整的局部名单和精选样本。二者必须区分。

接近完整的局部名单通常包括明确赛事、明确版本和明确来源的列表，例如某届 AFC U17 或 U23 的最终报名名单。即便如此，它也只代表该赛事和该版本，不代表同年龄段全部球员。

精选样本库用于建立研究入口和代表性案例，例如留洋历史、当前海外注册、青训项目专题、五大联赛亚洲教练。它们不是官方全量库，不能用 `verified_records`、`featured_records` 或站内卡片数量直接推导某国真实留洋人数或完整历史人数。

## 留洋样本边界

当前留洋样本不是官方全量人数。它按可核验公开资料整理代表性记录，并按 bucket 拆分路径：

- `big-five`：五大联赛顶级联赛一线队正式出场或主线样本。
- `big-five-youth`：五大联赛国家的青训、梯队、预备队或培养链。
- `big-five-lower-tier`：五大联赛国家的低级别职业联赛。
- `europe-other`：欧洲非五大联赛或非五大国家路径。
- `asia-other`：亚洲其他职业体系。
- `americas-other`：美洲或北美职业/预备队体系。

签约未生效、试训、媒体传闻、短期训练营和未来观察池不应被混入当前海外注册或正式留洋人数。

## 纳入与排除

优先纳入：

- 公开可访问、可回查来源支持的球员和赛事记录。
- 有明确姓名、出生日期或注册身份可交叉验证的球员。
- 有明确赛事、赛季、俱乐部、联赛层级和时间范围的记录。
- 能帮助解释中国青训、亚洲青年赛事或留洋路径的专题样本。

暂不纳入或仅作为备注：

- 无稳定来源的传闻。
- 私人社交媒体截图里的孤立信息。
- 未成年人私人信息、联系方式、家庭信息和非公开注册信息。
- 无法消歧的同名同姓记录。
- 仅有商业宣传但无可核比赛、注册或官方公告支撑的履历。

## 状态词

常见状态含义：

- `verified`：有较强来源支撑，关键身份和事实已交叉确认。
- `in-progress`：已建档但仍在补来源、字段或边界说明。
- `watch` / `pending`：适合继续观察，尚不能并入确定样本。
- `completed`：赛事或专题阶段性结束，不等于数据永远不再更新。

状态是站内维护状态，不是官方认证等级。
