# 项目范围与非官方声明

更新时间：2026-06-28

本站是青训球员与足球研究资料的静态索引项目，用来整理公开来源中的球员、赛事、留洋、教练和专题线索。它不是中国足协、AFC、FIFA、俱乐部、学校或任何经纪机构的官方数据库。

## 当前统计什么

| 模块 | 当前范围 | 主要落点 |
| --- | --- | --- |
| U 系列球员 | 中国、日本、韩国 U17/U23 种子名单，中国 U20 和亚洲对照组逐步补齐。 | `data/raw/players/*.json` |
| 赛事 | AFC U17/U20/U23、FIFA 中国参赛档案、重点青年赛事。 | `data/raw/tournaments.json`、`data/raw/tournament-archive.json` |
| 当前中国留洋 | 已有公开来源支撑的现役海外注册或重点观察样本。 | `data/raw/players/china-overseas-current.json` |
| 留洋历史 | 中日韩留洋历史建档、五大联赛重点记录和分层口径。 | `data/raw/overseas-history.json` |
| 中超青年样本 | 2026 中超 U21/U23 和具备留洋经历的中国青年样本。 | `data/raw/players/china-csl-2026-youth.json` |
| 专题档案 | 董路足球小将、中国男足青年教练和研究专题。 | `data/raw/dossiers.json`、`data/raw/china-men-youth-coaches.json` |
| 亚洲教练 | 五大联赛顶级联赛亚洲教练主口径及边界样本。 | `data/raw/big-five-asian-coaches.json` |

## 当前不统计什么

- 不承诺覆盖所有中国青训球员、所有海外球员或所有赛事名单。
- 不把当前样本数量解释为官方全量人数。
- 不把 Transfermarkt、Wikipedia、媒体报道或短视频线索单独当作官方事实。
- 不收录非公开个人信息、联系方式、家庭地址、证件信息或私人社媒内容。
- 不维护实时比分、实时转会、实时身价或商业级数据 API。
- 不将未生效转会、试训、传闻或观察线索提前计入当前注册。

## 样本库 vs 全量库

当前项目是样本库和研究索引，不是全量库。即使某个页面显示“当前中国留洋样本 12 人”，也只代表仓库中已经结构化、通过来源复核并纳入当前口径的样本，不代表官方当前留洋总人数。

全量名单只有在满足以下条件时才可标注为完整：

- 有官方报名表、注册名单或赛事最终名单作为基准。
- 能说明名单版本、发布日期和核查日期。
- 已把缺席、退赛、补招、未来生效转会和观察池分开。
- 覆盖矩阵中明确写为完成或有可验收的完成标准。

## 口径边界

当前注册优先看俱乐部、联赛或赛事官方注册。未来生效转会、试训、入队传闻、媒体预测和青训背景不改变 `registration_club`，除非已有当前有效注册来源。

赛事名单应区分：

- 官方终报名名单。
- 当前集训或征召名单。
- 观察池或专题追踪名单。
- 已退出、未使用或状态未知样本。

留洋统计应区分：

- 当前海外注册。
- 未来生效合同。
- 试训或训练观察。
- 已回流。
- 历史留洋记录。

教练统计应区分：

- 五大联赛顶级联赛。
- 杯赛、附加赛和青年队。
- 欧洲非五大、AFC 国字号和亚洲顶级联赛。
- AFC 成员协会主口径和土耳其、以色列等广义边界样本。
